    وقتی که برنامه مدارس را باز می کنیم پیغام می دهد که به HELPER مراجعه کنید و با بازکردن آن و ذخیره پیغام با SQLEXPRESS نمی توان ارتباط برقرار کردروبرو می شوم برای رفع مشکل چه باید کرد.

    یکی از پیامهای خطای سیستم دانش آموزی مربوط به سرویس های نرم افزار SQL server می باشد. در زیر به آموزش بازسازی این سرویس ها می پردازیم.

    ۱- روی my computer کلیک راست نموده و گزینه manage  را انتخاب کرده

    ۲- از گزینه های دست چپ، آخرین گزینه یعنی service and application  را انتخاب کرده (شماره ۱)و پیام ظاهر شده را تأیید کرده سپس در کادر وسط (شماره ۲) روی service کلیک کرده. لیست نرم افزارها و سرویسهای موجود در سیستم را نمایش می دهد.

    ۳- سرویس های SQL server را پیدا کرده و روی اولی و سومی و چهارمی آن یک بار کلیک کرده، سپس در سمت چپ روی گزینه restart یا start کلیک کرده تا رستارت شوند و مجدداً به حالت اجرا درآیند.

    اگر مشکل از این طریق حل نشد، باید از طریق کنترل پنل نرم افزار SQL server را حذف و دوباره نصب نمایید
